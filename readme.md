# Luminous Garden

**Luminous Garden** is a modern full-stack plant e-commerce platform built as a featured portfolio project. It offers a refined shopping experience for customers along with powerful management tools for sellers and administrators.

Designed with a production-focused mindset, the project combines a responsive frontend, secure payment integration, role-based dashboards, inventory management, and order tracking into one complete system.

## Live Demo

**Website:** https://luminous-garden.web.app

## Overview

Luminous Garden was built to demonstrate real-world full-stack development skills through a complete online marketplace. The platform supports customer browsing, secure checkout, online and offline payment flows, protected dashboards, and backend business logic for order and inventory management.

## Key Features

- Modern plant marketplace with responsive UI
- Product listing and detailed product pages
- Secure checkout flow
- Online payments with **Stripe** and **SSLCommerz**
- **Cash on Delivery** support
- Order tracking system
- Role-based authentication and authorization
- Separate dashboards for **Customer**, **Seller**, and **Admin**
- Seller request and approval workflow
- Inventory and product management
- Protected routes and backend API integration
